# KBank Manager

이 프로젝트는 Spring Boot와 Maven을 사용하여 구축된 KBank FEP 솔루션 관리 애플리케이션입니다.

## 요구 사항

- Java 17 이상
- Maven 3.6 이상

## 빌드 및 실행

프로젝트를 빌드하려면:

```bash
mvn clean install
```

애플리케이션을 실행하려면:

```bash
mvn spring-boot:run
```

## 구조

- `src/main/java`: Java 소스 코드
- `src/main/resources`: 리소스 파일
- `src/test/java`: 테스트 코드
